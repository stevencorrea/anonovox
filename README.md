# anonovox

To install dependencies:

```bash
bun install
```

To run in dev mode with live reloads:

```bash
bun --hot index.ts
```

To run in production mode:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.8. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
