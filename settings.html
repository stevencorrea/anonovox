Setup instructions
  1. In Azure Portal, create an Azure Bot resource and note its App ID and Secret.
  2. Set TEAMS_APP_ID and TEAMS_APP_SECRET on the server, then restart the app.
  3. Use the exact messaging endpoint shown above for the bot configuration.
  4. Enable the Microsoft Teams channel on the bot, then download the app package from this page and upload it in Teams.
  5. Find your Tenant ID in Azure Portal → Azure Active Directory → Overview and enter it above, or connect through Microsoft SSO below.
